# Support

## How to get help

### Questions and Discussions

If you have questions about how to use TraceForge, please:

1.  Check the [Documentation](docs/).
2.  Search existing [Issues](https://github.com/traceforge/traceforge/issues) to see if someone else has had the same problem.
3.  Start a discussion in [GitHub Discussions](https://github.com/traceforge/traceforge/discussions) (if enabled).

### Reporting Bugs

If you believe you have found a bug, please open an issue using the [Bug Report template](.github/ISSUE_TEMPLATE/bug_report.md).

### Feature Requests

If you have an idea for a new feature, please open an issue using the [Feature Request template](.github/ISSUE_TEMPLATE/feature_request.md).

## Commercial Support

Currently, TraceForge is a community-driven project. There is no official commercial support available at this time.
